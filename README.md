Запуск программы
1. Установка зависимостей в папке проекта:
pip install vk-api aiohttp flask flask-cors requests g4f

2. Настройка VK бота
В файле bot.py укажите данные:
VK_TOKEN = "токен_группы_вк"
GROUP_ID = 123456789  # ID вашей группы

3. Запуск (два компонента)
Веб-интерфейс (карта рисков):
В терминале написать: python backend.py
Откройте браузер: http://localhost:5000

VK бот (второй терминал):
Написать: python bot.py
